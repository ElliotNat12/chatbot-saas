Process

Mon process — Nouveau client

1
Analyser 5 min
  • Ouvrir /admin/ → onglet Nouveau client
  • Coller l'URL du site client → cliquer Analyser le site
  • Vérifier les champs auto-remplis : nom, bot, couleur, téléphone
  • Ajuster la FAQ si nécessaire (section "Base de connaissances")
  • Cliquer Déployer le chatbot → copier le lien démo généré
2
Enrichir 2 min
  • Aller dans Tester & évaluer → sélectionner le client
  • Cliquer 📋 Questionnaire client → copier le lien
  • Envoyer au client par SMS / WhatsApp avec le message type :
Message questionnaire
Bonjour ! Pour que votre chatbot réponde encore mieux à vos clients, pourriez-vous remplir ce petit formulaire (5 min) ? https://chatbot-saas-nine.vercel.app/api/onboarding-form?slug=[SLUG] Merci d'avance !
3
Livrer 3 min
  • Envoyer le lien de démo : https://chatbot-saas-nine.vercel.app/demo-[SLUG]/
  • Envoyer les 2 lignes de code à coller sur le site client
  • Préciser de les coller juste avant le </body>
Message de livraison
Bonjour [Prénom], Votre chatbot est en ligne ! Testez-le ici : https://chatbot-saas-nine.vercel.app/demo-[SLUG]/ Pour l'intégrer sur votre site, collez ces 2 lignes juste avant le </body> : <script src="https://chatbot-saas-nine.vercel.app/widget/chatbot.js"></script> <script src="https://chatbot-saas-nine.vercel.app/demo-[SLUG]/config.js"></script> Si vous ne gérez pas votre site, transmettez-les à votre développeur.
4
Suivre hebdomadaire
  • Chaque lundi → ouvrir Tester & évaluer
  • Vérifier le score chatbot (objectif > 7/10)
  • Consulter les suggestions d'amélioration générées automatiquement
  • Approuver / modifier / rejeter chaque suggestion
  • Consulter les questions sans réponse et les ajouter à la FAQ
  • Le rapport hebdo arrive par email chaque lundi à 7h

A — Bugs fréquents et solutions troubleshooting
Chatbot ne répond pas / erreur dans le chat
Vérifier ANTHROPIC_API_KEY sur Vercel → Settings → Environment Variables. La clé doit commencer par sk-ant-. Après modification : redéployer.
WhatsApp non reçu
Vérifier TWILIO_ACCOUNT_SID, TWILIO_AUTH_TOKEN, et TWILIO_WHATSAPP_TO (format exact : whatsapp:+336XXXXXXXX). Tester sur console.twilio.com.
Déploiement client échoué (erreur GitHub)
GITHUB_TOKEN expiré. Regénérer sur github.com → Settings → Developer settings → Personal access tokens → Tokens (classic). Scopes requis : contents: write.
FAQ vide dans "Suggestions" ou score 0/10
Soit la FAQ n'est pas encore remplie (aller dans Éditeur FAQ → vérifier), soit il n'y a pas encore de conversations loggées. Attendre 7 jours ou tester avec de vrais messages.
Email de notification non reçu
Vérifier RESEND_API_KEY. L'expéditeur doit être un domaine vérifié ou onboarding@resend.dev (test only). Vérifier les spams.
Dashboard admin inaccessible (401 au login)
Le mot de passe saisi doit correspondre exactement à ADMIN_SECRET dans Vercel env vars. Vérifier les espaces et la casse.
B — Ajouter des infos manuellement à la FAQ FAQ

Procédure

  • Ouvrir /admin/ → onglet Tester & évaluer
  • Sélectionner le client dans le menu déroulant
  • Descendre jusqu'à la section Éditeur FAQ
  • Ajouter le texte à la fin (format libre, le bot comprend tout)
  • Cliquer Sauvegarder & redéployer
  • Attendre ~1 minute → tester sur la démo pour confirmer

Format recommandé

Le format libre fonctionne, mais ce format structuré est optimal :

FAQ Question : Acceptez-vous les groupes scolaires ? Réponse : Oui, nous accueillons les groupes à partir de 10 personnes sur réservation. CONDITIONS Arrhes : 30% à la réservation, solde le jour J.

Le bot s'adapte à n'importe quel format — titres, listes, phrases, etc.

E — Commandes utiles CLI

Tester le rapport hebdomadaire manuellement

curl
curl "https://chatbot-saas-nine.vercel.app/api/weekly-report?secret=VOTRE_REPORT_SECRET"

Forcer la génération de suggestions pour un client

curl
curl -X POST "https://chatbot-saas-nine.vercel.app/api/suggest-improvements" \ -H "x-admin-secret: VOTRE_ADMIN_SECRET" \ -H "Content-Type: application/json" \ -d '{"slug":"nom-du-client","businessName":"Nom Complet","force":true}'

Générer les suggestions pour tous les clients

curl
curl "https://chatbot-saas-nine.vercel.app/api/suggest-improvements?secret=VOTRE_ADMIN_SECRET&auto=true&allClients=true"
